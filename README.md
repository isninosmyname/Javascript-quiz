# Javascript-quiz

Let's be honest, JavaScript can be weird. (literally) I built this project as a fun way to test developer knowledge while practicing how to build clean, scalable React applications.

## Technologies

In this project, I used a modern stack that prioritizes performance and a smooth developer experience:

* **React:** I use it a lot of times when I like.
* **Material UI:** To be able to get icons and writing CSS without havin to create a CSS file.
* **Zustand:** I was learning zustand. and its also better than Redux

---

## Getting Started

1. Clone the repo:
   `git clone https://github.com/your-username/javascript-quiz.git`

2. Install dependencies:
   `npm install`

3. Run the dev server:
   `npm run dev`
